Major League Baseball: Advanced SQL Insights

Built using data from the Maven Analytics SQL course

📌 Excerpt

Explored Major League Baseball datasets using advanced SQL techniques such as window functions, CTEs, and subqueries. Answered real-world analytical questions about schools producing players, team salaries, career spans, batting profiles, and player physical trends over time.

📖 Description

This project was built using advanced SQL concepts to extract insights from Major League Baseball datasets. It covers a wide range of questions that simulate real business use cases for data analysts. Key analysis areas include:

  • Trends in MLB player origins by school and decade
  • Team salary spending patterns and cumulative trends
  • Player career analysis: age at debut and retirement, longest careers, team loyalty
  • TDemographic insights like batting styles, shared birthdays, and changes in player physical attributes over time

Using SQL alone, I analyzed large-scale MLB datasets to reveal insights across a range of topics including player origins, career spans, spending patterns, and more.
